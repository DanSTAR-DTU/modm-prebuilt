Detailed Description

Basic Timer 6.

Interrupt handler:

MODM_ISR(TIM6)
{
Timer6::resetInterruptFlags(Timer6::FLAG_UPDATE);
...
}

For the STM32F2xx and STM32F4xx:

MODM_ISR(TIM6_DAC)
{
Timer6::resetInterruptFlags(Timer6::FLAG_UPDATE);
...
}
