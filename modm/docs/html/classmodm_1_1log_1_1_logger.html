Detailed Description

Interface to the Logger.

It is used by the macros defined below. This class overloads the shift-operator so that it is possible to write different message types to the logger.

Output forwarding.

We must use modm_always_inline here to prevent the generation of specialized functions for every type. Especially for strings this might cause a lot of code size bloat.

Example without modm_always_inline or only inline:

$ scons symbols | grep "Logger"
...
