Use bank as identifier filter for standard or extended messages.

The message have to match the given identifier exactly! For extended messages only the upper 14-bit are checked. The lower 15-bit are don't care!

Parameters
bankNumber of the filter bank
fifoFIFO assignment (message will be stored in Fifo 0 or Fifo 1)
id111 or 29-bit identifier
