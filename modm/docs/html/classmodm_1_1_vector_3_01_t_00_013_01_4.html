Detailed Description

template<typename T>
class modm::Vector< T, 3 >

Class for handling common vector operations (3D)

  • : addition of points
  • : different of points
  • : dot product or scalar multiplication / : scalar division ^ : cross product (determinant)

Adapted from the implementation of Gaspard Petit (gaspa.nosp@m.rdpe.nosp@m.tit@g.nosp@m.mail.nosp@m..com).
