Detailed Description

template<typename T>
class modm::Vector< T, 4 >

Class for handling common vector operations (4D)

  • : addition of points
  • : different of points
  • : dot product or scalar multiplication / : scalar division

Adapted from the implementation of Gaspard Petit (gaspa.nosp@m.rdpe.nosp@m.tit@g.nosp@m.mail.nosp@m..com).
