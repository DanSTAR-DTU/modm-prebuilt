Detailed Description

template<class Clock, class TimestampType = modm::Timestamp>
class modm::GenericTimeout< Clock, TimestampType >

Generic software timeout class for variable timebase and timestamp width.

See also
GenericPeriodicTimer
Template Parameters
ClockUsed clock which inherits from modm::Clock, may have a variable timebase.
TimestampTypeUsed timestamp which is compatible with the chosen Clock.
