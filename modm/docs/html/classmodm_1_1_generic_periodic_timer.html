Detailed Description

template<class Clock, typename TimestampType = modm::Timestamp>
class modm::GenericPeriodicTimer< Clock, TimestampType >

Generic periodic software timeout class for variable timebase and timestamp width.

See also
GenericTimeout
Template Parameters
ClockUsed clock which inherits from modm::Clock, may have a variable timebase.
TimestampTypeUsed timestamp which is compatible with the chosen Clock.
